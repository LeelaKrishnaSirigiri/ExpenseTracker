Personal Expense Tracker – Streamlit Web App

Description:
The Personal Expense Tracker is an interactive web application built using Python, Streamlit, Pandas, and Plotly. It allows users to efficiently manage and track their daily expenses. Users can add expenses, view all transactions, see total spending, and analyze category-wise spending using dynamic charts. This project demonstrates object-oriented programming (OOP) principles combined with a modern, interactive web interface.

Key Features:

Add and categorize daily expenses

View all recorded expenses in a dynamic table

Calculate total spending

Visualize spending by category using pie and bar charts

Dashboard with key metrics: total spending, number of transactions, and category count

Interactive and responsive UI using Streamlit

Tech Stack:

Backend: Python (OOP)

Frontend / UI: Streamlit

Data Handling: Pandas

Visualization: Plotly

Why This Project is Useful:

Helps users track personal finances easily

Provides insights into spending patterns

Demonstrates the integration of OOP with interactive web apps

Perfect for showcasing on GitHub or deploying on Streamlit Cloud
